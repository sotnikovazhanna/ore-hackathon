# OreScope AI — финальная версия приложения

Локальное Streamlit-приложение для предварительного автоматического анализа OM-изображений полированных шлифов.

## Что делает приложение

- принимает TIFF, PNG и JPEG, включая панорамы высокого разрешения;
- безопасно уменьшает огромные изображения до рабочего разрешения без `DecompressionBombError`;
- обрабатывает изображения тайлами и пакетами;
- оценивает долю зоны оталькования;
- оценивает долю сульфидной фазы;
- разделяет обнаруженные сульфидные участки на обычные и тонкие срастания;
- применяет экспертную логику:
  - тальк > 10% → оталькованная руда;
  - иначе преобладают обычные срастания → рядовая руда;
  - иначе → труднообогатимая руда;
- показывает интерпретируемую цветовую маску;
- формирует CSV, JSON, PNG, PDF и общий ZIP;
- поддерживает пакетную обработку;
- автоматически уточняет пограничные результаты в более высоком разрешении.

## Что изменено в финальной версии

- полностью убран вспомогательный `talc gate` из пользовательского интерфейса и финального инференса;
- интерфейс переписан для специалиста без знаний ML;
- технические параметры скрыты;
- добавлена понятная оценка надёжности предварительного заключения;
- сохранена официальная логика классификации по доле талька;
- добавлена автоматическая повторная проверка пограничных случаев;
- сохранена обработка панорам до 512 МБ на файл;
- сохранены пакетная обработка, PDF и ZIP-экспорт;
- добавлена проверка совместимости файлов перед запуском.

## Важное ограничение

Текущая реализация ordinary/fine остаётся хакатонным MVP:

1. кандидаты сульфидной фазы выделяются быстрым алгоритмом по отражательной способности;
2. тип срастаний определяется нейросетью по локальным тайлам;
3. это не является полноценной объектной сегментацией каждого отдельного включения.

Сегментация талька выполнена U-Net с ResNet18 по ручным proxy-маскам.

Зафиксированные test-метрики модели талька:

- Mean IoU: 0.4517;
- Dice: 0.6054;
- MAE площади: 10.27 процентного пункта;
- F1 решения `тальк > 10%`: 0.9091;
- Precision: 1.0000;
- Recall: 0.8333.

Ordinary/fine:

- Macro F1: около 0.833;
- ROC-AUC: около 0.876.

Требования ±3% и F1 ≥ 0.90 полностью автоматически пока не достигнуты. В интерфейсе результат честно называется предварительным и сопровождается визуальной маской.

## Установка поверх текущего проекта

### 1. Остановить Streamlit

```powershell
Ctrl + C
```

### 2. Распаковать архив финальной версии

Запустить установщик из распакованной папки:

```powershell
powershell -ExecutionPolicy Bypass -File .\install_release.ps1 `
  -ProjectRoot "C:\Users\Мария\ore_hackathon" `
  -InstallRequirements
```

Установщик:

- создаст резервную копию текущих `app.py`, `src/`, `.streamlit/`;
- скопирует согласованный набор файлов;
- удалит старый Python-кэш;
- запустит автоматическую проверку.

Успешный результат:

```text
VERIFY OK
FINAL RELEASE INSTALLED AND VERIFIED
```

### 3. Проверить модели

В проекте должны находиться:

```text
models/
├── ordinary_fine_mobilenet_v3_small.pth
└── talc_unet_resnet18.pth
```

### 4. Запустить

```powershell
cd "C:\Users\Мария\ore_hackathon"
.\.venv\Scripts\python.exe -m streamlit run app.py
```

## Как пользоваться

1. Открыть `http://localhost:8501`.
2. Выбрать режим **«Автоматически — рекомендуется»**.
3. Загрузить одну панораму или несколько файлов.
4. Нажать **«Начать анализ»**.
5. Проверить:
   - предварительный класс руды;
   - долю зоны оталькования;
   - долю сульфидной фазы;
   - ordinary/fine;
   - надёжность;
   - цветовую маску.
6. Скачать PDF или общий ZIP.

## Режимы

- **Автоматически** — оптимальный баланс скорости и детализации; пограничные результаты уточняются повторно.
- **Быстро** — минимальное рабочее разрешение для предварительного просмотра.
- **Подробно** — больше деталей и большее время обработки.

## Проверка скорости панорамы

```powershell
.\.venv\Scripts\python.exe tools\benchmark_panorama.py `
  "C:\path\to\panorama.jpg" `
  --max-mp 10
```

Для примера 344 MP ранее были безопасно уменьшены до 6 MP и обработаны на CPU примерно за 19 секунд. Фактическое время зависит от оборудования и выбранного режима.
