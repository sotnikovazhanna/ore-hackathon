# Чек-лист финальной загрузки

## Приложение

- [ ] Приложение запускается после чистого `git clone` по README.
- [ ] В папке `models/` есть обе обязательные модели.
- [ ] Большой JPEG 120+ МБ загружается без ошибки Pillow.
- [ ] TIFF, PNG и JPEG проходят обработку.
- [ ] Пакет из двух панорам обрабатывается без падения.
- [ ] Время одной контрольной панорамы меньше 5 минут.
- [ ] При доле талька >10% итог всегда «Оталькованная руда».
- [ ] PDF, CSV, JSON, PNG и общий ZIP скачиваются.
- [ ] На главном экране нет `talc gate`, названий архитектур, batch size и ML-порогов.

## Демонстрация

- [ ] Рядовая руда.
- [ ] Труднообогатимая руда.
- [ ] Оталькованная руда.
- [ ] Большая панорама.
- [ ] Пакетная обработка.
- [ ] PDF-отчёт.

## Честные формулировки

- Система формирует предварительное автоматическое заключение.
- Маска талька обучена по ручным proxy-маскам на основе слабой экспертной разметки.
- Automatic area MAE на test — 10.27 п.п.; требование ±3% пока не достигнуто.
- Ordinary/fine Macro F1 — около 0.833; требование 0.90 пока не достигнуто.
- Продукт поддерживает локальный запуск и визуальную экспертную проверку.
