# Шаг 4 — сегментация областей оталькования

1. Скопируйте `04_train_talc_segmenter.ipynb` в корень проекта.
2. Дополнительные пакеты обычно не нужны: используются зависимости,
   установленные на предыдущих шагах.
3. Откройте ноутбук, выберите `.venv (Python 3.11)` и нажмите `Run All`.

Результаты появятся в `outputs/talc_segmenter`, модель — в `models`.
