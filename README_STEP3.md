# Шаг 3

1. Скопируйте `03_train_intergrowth_classifier.ipynb` в корень проекта.
2. Установите дополнительные зависимости:

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements_step3.txt
```

3. Откройте ноутбук, выберите kernel `.venv` и нажмите `Run All`.

Обучение на CPU может занять заметное время. На NVIDIA GPU оно будет быстрее.
