# Шаг 2

Скопируйте `02_prepare_training_data.ipynb` в корень проекта рядом с
`01_data_audit.ipynb`.

Перед запуском убедитесь, что существует папка `outputs/audit`,
созданная первым ноутбуком.

Откройте ноутбук, выберите kernel `.venv` и нажмите `Run All`.

Результаты будут сохранены в `data/processed`.
